# getTwitFollow

For ubuntu

git clone https://github.com/milkmoney10/getTwitFollow.git
<br>
cd getTwitFollow
<br>
source bin/activate
<br>
pip install -r requirements.txt
<br>
<br>
Add your keys/tokens and Twitter handles you want to look up
<br>
<br>
run with: python3 twitFollowNoUi.py
<br>
